# Fret.io 🎸

> **A high-fidelity retro Android application designed to elevate your guitar mechanics and fretboard mastery.** Built with Jetpack Compose, featuring vintage-inspired monochrome aesthetics, zero-rounded geometry, and state-of-the-art interactive training modules.

---

## 📻 Visual Design & Philosophy

Fret.io is styled after classic Poolsuite aesthetics and vintage tech interfaces. 
* **Palette**: Sleek shades of monochrome grey (`#E5E5E5` and `#CCCCCC`) resting on a warm, organic sand beige canvas (`#FAF6F0`).
* **Geometry**: Sharp, blocky, zero-rounded outlines (`0.dp` corner radius) and thick physical-feeling board borders that respond elegantly to touch.
* **Micro-Animations**: Subtle tap offsets and accent transitions that make every button and dial feel like clicky physical hardware.

---

## 🕹️ Interactive Features

* **Fretboard Explorer**: Dynamically highlight scale intervals, chord formulations, and degree structures across the neck in any tuning.
* **Chord Library**: Look up complex voicings on a pristine canvas with responsive interval degree annotations on each string.
* **Ear Training Hub**: Custom audio engine playing high-fidelity pitches to train your interval spelling and degree recall.
* **Theory Arcade**: Speedrun scale degrees under pressure to build split-second fretboard reflexes.
* **Resources Screen**: A fully reactive search index with category toggles for tabs, PDF charts, and masterclass lessons.
* **Metronome & Tuner**: Clean, clicky timekeeping tools and high-fidelity pitch detection helpers.

---

## 🛠️ Technology Stack

* **UI Framework**: Modern Jetpack Compose (Declarative UI)
* **Design Guidelines**: Material 3 adapted for custom blocky Poolsuite parameters
* **Audio Layer**: Native Kotlin/Java AudioTrack synthesizing clean waveforms and playing PCM samples
* **Compilation**: Kotlin JVM toolchain 21 with Gradle desugaring configured for robust device compatibility

---

## 🚀 Getting Started

1. Clone this repository:
   ```bash

   ```
2. Open in **Android Studio (Ladybug or newer)**.
3. Sync Gradle and run on an emulator or physical device.
